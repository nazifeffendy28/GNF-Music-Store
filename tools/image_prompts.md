# Panduan Generate Gambar Produk (Higgsfield AI)

Setiap produk memiliki `gambar_url` di database. Letakkan file hasil generate di folder `assets/` mengikuti path tersebut (contoh: `assets/images/gitar-fender-strat.jpg`).
Aplikasi otomatis memakai gambar asli bila filenya ada; jika tidak, dipakai **placeholder SVG** bertema kategori yang sudah dibuat (`assets/placeholder.php`).

**Model rekomendasi:** `marketing_studio_image` (foto produk komersial), aspect ratio `1:1`.
